# 哈希表

哈希表的使用场景一般是查找一个元素是否出现在集合里面

# 哈希函数

使用哈希函数完成映射，达到在O(1)时间内找到结果。此时为了保证映射出来的索引数值都落在哈希表上，我们会在再次对数值做一个取模的操作。

# 哈希冲突

如果两个关键字的映射为同一个结果称为哈希冲突。

## 拉链法

将冲突的结果存放在链表中，通过链表的索引找到函数即可
